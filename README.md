# pePpe

Un motore fisico in 2D realizzato in Java.

Una versione completa è presente in: https://github.com/giuseppe-gitHub/pePpe/blob/master/pePpe-full.rar

Per provare le simulazioni presenti, scompattare l'archivio pePpe-full.rar ed eseguire il file "consoleTestSuit.bat" (in sistemi Windows) oppure il file "consoleTestSuit.sh" (in sistemi Linux).

Una volta scelta la simulazione ed impostato i parametri (se necessario), far partire la simulazione premendo il tasto 'p' sulla schermata grafica.
Premere il tasto 'h' sulla schermata grafica per visualizzare le funzionalità presenti.
